Ongoing & past Projects

My current personal project is developing a distributed hierarchical Model Predictive Control framework

Most modern humanoids utilise an MPC for higher level planning and a whole body controller for low level execution. While this hierarchical architecture has been proven to work well, MPCs are inherently conputationally intensive and model specific.

Inspired by Amatucci et al. [1], the goal of this personal project is to develop distributed MPC framework for humanoid application
